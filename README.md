# Fact Checking AI — Privacy Policy

This repository publishes the privacy policy for the Fact Checking AI Assistant browser extension.

Published URLs:

- https://fuctcheckingai.github.io/fact-checking-privacy/
- https://fuctcheckingai.github.io/fact-checking-privacy/privacy.html

## Publishing

In the GitHub repository settings, open **Pages**, select **Deploy from a branch**, choose the `main` branch and the `/ (root)` folder.

Keep `index.html` and `privacy.html` identical when updating the policy.
