Privacy Policy for Fact Checking AI Assistant

Last updated: June 10, 2026

Fact Checking AI Assistant helps users check selected text against public evidence sources and optional AI providers.

Data the Extension Handles

The extension may process text that you select, paste, or submit for fact checking; extension settings; short-lived runtime state and cached evidence results; and payment status when payments are enabled.

How Data Is Used

Selected or submitted text is used only to generate a fact-checking result, collect supporting public sources, and format the answer shown to you. Settings configure extension behavior. Evidence cache is stored locally to reduce repeated API calls.

Third-Party Services

Depending on your settings and the claim being checked, text or search queries may be sent to Pollinations AI, ApiFreeLLM, GDELT, Wikipedia, Wikidata, OpenAlex, Semantic Scholar, Crossref, arXiv, DuckDuckGo Instant Answer API, PubMed / NCBI E-utilities, Google Knowledge Graph Search, or a user-provided SearXNG instance.

These services may process requests according to their own privacy policies. Do not submit sensitive personal information unless you are comfortable sending it to the selected services.

Storage

The extension uses chrome.storage.sync for non-sensitive user settings and chrome.storage.local for API keys, selected text, runtime state, and temporary evidence cache.

Data Sharing and Sale

The extension does not sell user data. The extension does not share user data for advertising, profiling, or unrelated analytics. Data is sent to third-party services only to provide fact-checking, source retrieval, AI response generation, or payment verification when enabled.

API Keys

If you enter API keys in the settings page, they are stored locally in your Chrome profile (not synchronized by the extension) and used only to call the corresponding service.

The extension includes a shared, publicly extractable Google Knowledge Graph browser key so the default evidence search works without user setup. You may replace it with your own locally stored key.

Payments

If payments are enabled, payment status may be checked through the configured payment provider. Payment details are handled by the payment provider, not by this extension.

Permissions

The extension requests permissions needed to read selected text when you invoke the checker, show a context-menu action, inject the checker overlay into the active tab, store settings and cached evidence, and contact fact-checking, source, AI, and payment-related services.

User Control

You can delete extension data by removing the extension from Chrome or clearing extension storage in the browser profile.

Contact

For privacy questions, open a support request at github.com/fuctcheckingai/fact-checking-privacy/issues.
